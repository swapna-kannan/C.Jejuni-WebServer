Introduction

FbP server features a full genomics pipeline for analysis of foodborne bacterial isolates. It includes genome assembly, gene prediction, functional annotation, & comparative genomics pipeline stages. Any contiguous range of pipeline stages can be run on a set of isolates. The comparative genomics stage features some analyses that will only function if multiple isolates are provided.

Pipeline Details

This pipeline runs four steps - Genome Assembly, Gene Prediction, Functional Annotations and Comparative Genomics.
