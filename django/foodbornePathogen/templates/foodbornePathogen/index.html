Introduction

The Foodborne Pathogen Webserver features a full genomics pipeline for analysis of foodborne Campylobacter jejuni   isolates. It includes genome assembly, gene prediction, functional annotation, & comparative genomics pipeline stages. To run the pipeline, please click the Run Pipeline button.

Pipeline Details

This pipeline runs four steps - Genome Assembly, Gene Prediction, Functional Annotations and Comparative Genomics.
