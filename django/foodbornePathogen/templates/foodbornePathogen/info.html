Pipeline Information

This foodborne pathogen analysis pipeline performs genomics analysis on C. jejuni isolates. The full pipeline contains these four steps: genome analysis, gene prediction, functional annotation, and comparative genomics. The pipeline can be run in full, as individual steps, or as any consecutive combination of steps.

Instructions

1. Access the pipeline steps you would like to run using the Run Pipeline button.

2. Enter your email and upload relavent files.

3. Select pipeline and analysis options.

4. Click submit. You will reach a submission confirmation page. A results page will be sent to the email provided with links to download analysis results.

For more information, please refer to our GitHub and Wiki pages.
