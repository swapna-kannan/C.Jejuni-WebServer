Pipeline Information

This foodborne pathogen analysis pipeline performs genomics analysis on Campylobacter jejuni isolates. The full pipeline contains four steps: genome analysis, gene prediction, functional annotation, and comparative genomics. The pipeline can be run in full, as individual steps, or as any set of consecutive of steps.

Instructions

  1. Access the pipeline steps you would like to run using the "Run Pipeline" button.
  2. Enter your email and upload required files.
    • Each isolate you wish to process must be uploaded as an individual .zip. You may upload multiple isolates.
    • Each .zip must contain all dependencies required for that pipeline segment as shown in the table below.
    • File extensions must match those in the table exactly.
    • Parenthesized inputs are optional depending on whether the user has requested the corresponding outputs shown in the data dependecy chart below.
    • All files must have the same basename as the zip file containing them, with the exception of FASTQ files.
    • Paired-end read FASTQ files must be individually gunzipped as {isolateBasename}_1.fq.gz & {isolateBasename}_2.fq.gz, zipped into {isolateBasename}.zip which is zipped inside the isolate .zip.
    • If only FASTQ files are required, the redundant nested .zip folder should be eliminated.

    • Empty dummy files to exemplify that data upload for various jobs:
      • Full Pipeline
      • Gene Prediction
      • Functional Annotation
      • Comparative Genomics
  3. Select pipeline and analysis options.
  4. Click "Submit Job".
    • Depending on the size of your upload, it may take a few minutes before you are redirected to a submission confirmation page. Do NOT navigate away from this page until the redirect is complete.
    • An email will be sent to the address provided with a link to download analysis results.


    For more information on options, file types, and analysis methods, please refer to our GitHub and Wiki pages. A diagram of the pipeline is shown below.
